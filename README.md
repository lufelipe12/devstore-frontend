# Description

This repository was made for devnology fullstack test. Here's a front-end for user registration, login and shopping.

## Technologies

- TypeScript 
- React.js
- Styled-components
- React-hook-form
- React-cookie
- React-icons
- React-toastify
- Framer-motion
- Material UI
- Vercel for deploy

## Project Structure

## Installation

